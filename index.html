📚 TEMPORADAS ARQUIVADAS

Como funciona: cada jogo que você registra gera XP distribuído em 5 categorias (Outside, Inside, Playmaking, Defense, Athleticism). Você usa o XP da categoria correspondente para subir os atributos daquele grupo.

🌱 Multiplicador por TIER: em vez de comparar com sua média, agora você define seu tier de carreira no perfil. Rookies ganham até +60% XP (estão começando, evoluem rápido) e MVPs ganham −40% (já são consolidados, evoluir é brutal). Atualize seu tier quando seu personagem evoluir.

Custo de upgrade: calculado como peso do atributo × 5 × multiplicador do patamar atual. Atributos importantes (3PT, Speed, Dunk) custam mais. Subir de 70→71 é barato; subir de 90→91 é brutal.

📐 Overall ponderado por posição: o OVR não é mais média simples — cada posição valoriza atributos diferentes. PG pesa 3PT e Ball Handle; C pesa Block e Interior Defense. Mude sua posição no perfil pra recalcular.

🏆 Playoffs: marque o checkbox no formulário pra jogos de mata-mata. Ganha +20% XP no jogo.

📦 Compilado: se você joga 10/20/30 partidas em sequência e tem só a média, marque "compilado" e informe o nº de jogos. O sistema multiplica o XP do jogo pelos N partidas.

🏅 Marcos da temporada: bônus únicos de XP por feitos cumulativos (1º triple-double, 10/25/50 double-doubles, sequências de vitórias, jogos de 40+/50+/60+ pts, etc).

🏆 Prêmios individuais: na aba AWARDS, marque cada prêmio que você ganhar (MVP, DPOY, ROY, All-Star, All-NBA, All-Defensive Teams, etc) e ganhe um bônus de XP equilibrado pela importância de cada prêmio.

🎯 O QUE É

Dashboard pessoal de evolução de carreira no NBA 2K26. Você registra cada jogo (manual ou via screenshot+Gemini), ganha XP distribuído em 5 categorias, gasta XP pra subir seus 37 atributos seguindo um motor de custo que respeita o realismo da NBA real (envergadura, biotipo, arquétipo). Badges desbloqueiam automaticamente conforme os atributos sobem — espelhando o sistema do jogo.

⚡ FUNCIONALIDADES v5

  • Auto-save no arquivo (rede/NetApp/disco) via File System Access API
  • Multi-backend (localStorage + IndexedDB) com aviso visível se falhar
  • 15 build presets 2K26 (PG Slasher, SG 3&D, C Paint Beast…)
  • Motor de custo 4 fatores: base × dificuldade × biotipo × arquétipo
  • Wingspan/envergadura no perfil — afeta custo de Block/Steal/Rebound
  • Parser de stats (cola texto do Gemini) e parser de atributos (cola tela do builder)
  • 40 badges reais com lógica OR/AND/SINGLE + restrição de altura
  • Marcos manuais alternáveis (✋ MANUAL vs 🤖 AUTO)
  • OVR híbrido: cálculo automático ou cravado manualmente
  • Compilado V/D proporcional + DD/TD/highs manuais
  • Atalhos: Ctrl+S salva, Enter registra jogo, Tab navega

📚 FONTES OFICIAIS

  • Chart de badges: @JPforThreee (limiares reais por tier)
  • Categorias e descrições: @The2KCentral (imagem oficial)
  • 37 atributos: lista oficial do NBA 2K26 (Intangibles removido — é cosmético)
  • Caps por posição: pesquisa de community 2K (referência, sem bloqueio rígido)

🛠 STACK

  • HTML/CSS/JS standalone — 1 arquivo, zero dependências
  • Fonts: Bebas Neue · JetBrains Mono · Archivo (Google Fonts)
  • Storage: localStorage + IndexedDB + File System Access API
  • Tamanho: ~190KB · ~3500 linhas

👤 PROJETO PESSOAL

Feito para o jogador. Tudo roda offline, sem servidor, sem login. O save é seu — exporte JSON quando quiser. Use a versão de arquivo vinculado pra gravar direto na rede ou disco.

Filosofia: simplicidade pro usuário, complexidade nas fórmulas · nada deduzido errado · valores derivados, não chutados.

📋 COLAR STATS DO JOGO

Cole o texto extraído pelo Gemini (ou digite no formato livre). O parser entende vários formatos — listas, tabelas, JSON, frases corridas.

💡 1 jogo: "extraia as stats em chave:valor (PTS, REB, AST...)". Vários jogos (foto da tabela): peça ao Gemini "transcreva a tabela, UMA LINHA POR JOGO: DATA | OPP | W ou L | PLACAR | MIN | PTS | REB | AST | STL | BLK | TO | FGM-FGA" — cola aqui e o app registra todos de uma vez.
